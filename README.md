# dev-swiss

A Swiss Army knife CLI toolkit for developers.

## Installation

```bash
cargo install --path crates/dev-swiss
```

## Tools

| Command | Description | Docs |
|---------|-------------|------|
| `dev-swiss password` | Generate secure random passwords | [password.md](docs/password.md) |

## Development

```bash
cargo build        # Build
cargo test         # Run tests
cargo run -- help  # Show available commands
```
